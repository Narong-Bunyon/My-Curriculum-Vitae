Example Endpoints

A simple reference format I follow when building APIs.

Method Route Purpose Status
GET /api/users List users 200
POST /api/users Create user 201
GET /api/users/:id Get user detail 200 / 404
PATCH /api/users/:id Update user 200 / 400
DELETE /api/users/:id Remove user 204
Tip: Return consistent JSON response shapes (success, message, data, error).
