Lab Checklist

What I practice using Packet Tracer and CLI.

  • Build simple LAN (PC ↔ Switch ↔ Router)
  • Configure IP address + default gateway
  • Subnetting practice (different network sizes)
  • Create VLANs and assign switch ports
  • Inter-VLAN routing (router-on-a-stick)
  • Basic routing (static routes)
  • Troubleshooting with ping/traceroute
Tip: Always verify with show commands after each config.

CLI Commands Cheat Sheet

Quick reference I use during labs. (Switch/Router basics)

Basic

enable
configure terminal
hostname R1
no ip domain-lookup
copy run start

Interfaces

show ip interface brief
interface g0/0
ip address 192.168.1.1 255.255.255.0
no shutdown

Switch VLAN

show vlan brief
vlan 10
name SALES
interface f0/1
switchport mode access
switchport access vlan 10

Troubleshoot

ping 8.8.8.8
traceroute 8.8.8.8
show running-config
show ip route
